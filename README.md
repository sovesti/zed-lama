# Zed Lama extension

Adds [Lama](https://github.com/PLTools/Lama) language to [Zed](https://zed.dev/).

Built on [tree-sitter](https://tree-sitter.github.io/tree-sitter/index.html) grammar for Lama: <https://github.com/sovesti/tree-sitter-lama>.

## Installation

See Zed documentation: <https://zed.dev/docs/extensions/installing-extensions>.

## Features

- Syntax highlighting
- Bracket matching
- Compilation unit outline
- Auto-indentation
