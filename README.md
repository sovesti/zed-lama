# Zed Lama extension

Adds [Lama](https://github.com/PLTools/Lama) language to [Zed](https://zed.dev/).

Built on [tree-sitter](https://tree-sitter.github.io/tree-sitter/index.html) grammar for Lama: <https://github.com/sovesti/tree-sitter-lama>.

## Installation

- Clone this repository
- In the Zed's Extension view (Zed > Extensions or ```Ctrl + Shift + x```) choose "Install Dev Extension"
- Pick the folder with cloned repository

## Features

- Syntax highlighting
- Bracket matching
- Compilation unit outline
- Auto-indentation
